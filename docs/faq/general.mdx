# General FAQ

## Which section should I check first?

If you want to complete a task, check **How-to**. If something is not working as expected, check **Troubleshooting**.

## What should I do if I cannot find an answer?

Collect a screenshot, the exact error message, the product version, and the time when the issue occurred before escalating the case.

## Will these articles be updated?

Yes. Recurring support cases should be turned into new articles, and existing articles should be updated when the product or workflow changes.

## Can documents include PDF downloads?

Yes. Important guides can link to PDF files. A separate process can be added later to generate PDFs automatically.

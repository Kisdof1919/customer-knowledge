# Device Shows as Offline

## When to Use This Article

Use this article when a device appears offline in the system, or when the system cannot read the device status.

## Troubleshooting Steps

1. Confirm that the device is powered on.
2. Check whether Ethernet, Wi-Fi, or mobile network connectivity is available.
3. Verify that the device IP address or network configuration has not changed.
4. Confirm that firewall or router rules are not blocking the device from reaching the server.
5. Restart the device, wait 2 to 5 minutes, and refresh the device status.

## Information to Collect

- Device serial number
- Current network environment
- Time when the device first appeared offline
- Screenshot from the system
- Recent network or device configuration changes

## If the Issue Persists

Escalate the case with the information listed above.

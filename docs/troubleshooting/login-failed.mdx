# Login Fails

## When to Use This Article

Use this article when a user cannot sign in, sees an invalid credentials message, fails verification, or is signed out immediately after login.

## Troubleshooting Steps

1. Confirm that the login URL is correct.
2. Confirm that the username or email address is entered correctly.
3. Use **Forgot Password** to reset the password.
4. Clear the browser cache and try again.
5. Test in another browser or a private browsing window.
6. If multi-factor authentication is enabled, confirm that the authenticator device time is accurate.

## Common Issues

| Symptom | Likely cause | Resolution |
| --- | --- | --- |
| The system says the password is incorrect | The password is expired or entered incorrectly | Reset the password |
| The account is locked | Too many failed login attempts triggered a security policy | Wait for unlock or contact an administrator |
| The page is blank after login | Browser cache or extension conflict | Clear the cache or try another browser |

# Reset a Password

## When to Use This Article

Use this article when a user has forgotten their password, or when an administrator needs to help a user regain access.

## Prerequisites

- The user account is active.
- The administrator has user management permissions.

## Steps

1. Sign in to the admin portal.
2. Open the **User Management** page.
3. Search for the user who needs a password reset.
4. Select **Reset Password**.
5. Follow the on-screen instructions to send a reset link or set a temporary password.

## Expected Result

The user can sign in with the new password or reset link.

## Common Issues

| Symptom | Likely cause | Resolution |
| --- | --- | --- |
| The user does not receive the email | The email address is incorrect or the email was filtered | Check the email address and ask the user to review the spam folder |
| The reset link has expired | The link is past its validity period | Send a new reset link |

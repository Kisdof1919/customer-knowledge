# Export a Report

## When to Use This Article

Use this article when a customer needs to export system data for reporting, archiving, or sharing with another team.

## Prerequisites

- The account has permission to view and export reports.
- The correct date range and filters have been selected.

## Steps

1. Open the **Reports** page.
2. Select the report type.
3. Set the date range and filters.
4. Select **Export**.
5. Choose the export format, such as Excel or PDF.

## Expected Result

The system generates the report file and starts the download.

## Common Issues

| Symptom | Likely cause | Resolution |
| --- | --- | --- |
| The exported report is empty | The filters are too narrow | Expand the date range or remove some filters |
| The export button is not visible | The account does not have export permission | Ask an administrator to grant export permission |

# Customer Knowledge Base

This knowledge base organizes support cases into searchable customer-facing documentation. Each article should answer one clear question or solve one specific issue.

## Folder Structure

- **Getting Started**: first-time setup, basic configuration, and key concepts.
- **How-to**: task-based instructions, such as resetting a password or exporting a report.
- **Troubleshooting**: symptoms, likely causes, and resolution steps.
- **FAQ**: short answers to frequent questions.

## Maintenance Rules

1. One case should become one document.
2. Use customer-searchable titles, such as "Reset a password" or "Login fails with invalid credentials".
3. Prefer clear steps before background explanation.
4. Prioritize recurring cases first.
